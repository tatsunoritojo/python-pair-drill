# Python ペアドリル - 概要

ステータス: active

## 何を解決するか

Python3エンジニア認定基礎試験の「1対1対応」項目（構文・組み込み関数・型・演算子など、知っているか知らないかが直接得点に直結する暗記寄りの設問）を、繰り返し練習する手段が少ないという問題を解決する。

4択 + 即時フィードバック形式でスマホからすき間時間に練習でき、弱点優先モードで間違えた問題を集中的に再出題する。

## 誰のためか

Python3エンジニア認定基礎試験を受験予定の学習者。
主な利用者は東城さん自身だが、GitHub Pages で公開しているため他の受験者も利用可能。

## 主要概念

- **問題バンク**: `questions.js` / `logic-questions.js` / `concepts.js` に分割格納された4択問題集合。実行時に選択肢順・問題順をランダムシャッフル
- **弱点優先モード**: localStorage に蓄積された問題別正答率データを使い、不正解率の高い問題を重み付けして出題する
- **模擬試験モード**: `mock-exams/*.json` を読み込み、本番形式（問題数・時間制限）で通し練習する
- **統計・履歴**: 問題別の正答率と学習履歴を localStorage に端末保存。サーバへの送信なし
