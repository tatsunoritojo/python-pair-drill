# Python ペアドリル

Python3エンジニア認定基礎試験向けの「1対1対応」項目を 4択 + 即時フィードバックでドリルできるスマホ向けクイズアプリ。

## 機能

- 4択クイズ（選択肢順・問題順ともランダムシャッフル）
- カテゴリ絞り込み・出題数選択（10/20/全問）
- 弱点優先モード（過去の不正解率で重み付け出題）
- 即時フィードバック + 解説
- スコア・ストリーク（連続正解）・経過時間
- 結果画面で間違えた問題のみ再挑戦
- 問題別正答率の統計画面
- ダークモード（システム設定追従 + 手動切替）
- localStorage で履歴・統計を端末保存
- キーボードショートカット（1-4/A-D で選択肢、Enter で次へ）

## 構成

純粋な静的サイト。ビルド工程なし。

```
index.html
style.css
app.js
questions.js   問題データ
.nojekyll      Pages の Jekyll をスキップ
```

## 開発

ローカルで開く場合は `index.html` をブラウザで直接開くだけ。

問題を追加するには `questions.js` の `QUESTIONS` 配列に項目を足す。

```js
{
  id: 'unique-id',           // localStorage の統計キー
  category: 'カテゴリ名',
  question: '問題文',
  choices: ['正解', '誤答1', '誤答2', '誤答3'],
  correct: 0,                // choices の正解インデックス（実行時シャッフルされる）
  explanation: '解説',
}
```
